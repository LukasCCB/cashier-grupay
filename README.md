## Introduction

Laravel Cashier GruPay provides an expressive, fluent interface to [GruPay's](https://grupay.app) subscription billing services. It handles almost all of the boilerplate 
subscription billing code you are dreading writing. In addition to basic subscription management, Cashier can handle swapping subscription, subscription "quantities", subscription pausing, cancellation grace periods and much more.

## Official Documentation

Documentation for Cashier GruPay can be found on the [GruPay website](https://documentation.grupay.app).
